## 7. Разграничение прав доступа

*   **Менеджер:** Полный контроль над проектами, дефектами, пользователями, отчетами.
*   **Инженер:** Создание, обновление дефектов в своих проектах, добавление комментариев и вложений.
*   **Наблюдатель:** Просмотр всей информации, отчетов, без возможности внесения изменений.
